# Borutogotchi

Tamagotchi game in Boruto world!

## Installation

```
Use the package `mvn` to compile project.
```

Use this build commands:
```
    mvn compile
    mvn javafx:run
```

## How to play

```
    Create an account/or load existing acc
    Try to reach HOKAGE rang
    if your Health will be `0`, you will die)))
    Good luck, Have fun!
```
